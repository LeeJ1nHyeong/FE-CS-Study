# DOM에서의 id, class 속성

### id

* 태그에 **유일한** 이름을 붙이고 싶을 때 사용
* 전체 페이지에서 단 하나의 요소에만 지정 가능 -> 중복 불가
* CSS 적용 시 id명 앞에 #을 붙여서 사용

### class

* **같은 유형으로 반복**되는 태그들을 유형으로 분류할 때 사용
* 중복 가능
* CSS 적용 시 class명 앞에 .을 붙여서 사용
